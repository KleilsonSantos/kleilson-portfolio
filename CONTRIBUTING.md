# Contributing

Obrigado por considerar contribuir com o portfólio open source de Kleilson dos Santos.

## Como contribuir

1. Faça um fork do repositório
2. Crie uma branch: `feature/nome`, `fix/nome`, `docs/nome`
3. Siga [Conventional Commits](https://www.conventionalcommits.org/) com Gitmoji quando aplicável
4. Execute `npm run lint` e `npm run build` antes do PR
5. Abra um Pull Request descrevendo o contexto técnico

## Padrões

- Escopo mínimo por PR
- Textos de interface em pt-BR
- Não inventar informações profissionais — use apenas dados verificáveis
- Consulte `docs/adr/` antes de mudanças arquiteturais
