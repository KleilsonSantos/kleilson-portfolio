# Security Policy

## Versões suportadas

| Versão | Suportada |
|--------|-----------|
| 0.1.x  | ✅        |

## Reportar vulnerabilidade

**Não abra issues públicas para vulnerabilidades de segurança.**

Envie detalhes para: **kleilson@icloud.com**

Inclua:

- Descrição da vulnerabilidade
- Passos para reprodução
- Impacto potencial
- Sugestão de mitigação (se houver)

Responderemos em até 5 dias úteis.

## Escopo

- Código deste repositório
- Formulário de contato e APIs associadas (quando em produção)
- Pipeline CI/CD

## Práticas adotadas

- Sanitização de inputs no frontend
- Validação server-side (planejada Fase 3)
- Dependabot e CodeQL (planejados Fase 2+)
