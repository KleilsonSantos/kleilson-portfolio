# Support

Como obter ajuda com este projeto.

Este repositório é um **portfólio pessoal open source** — não há SLA de suporte comercial. Respostas acontecem quando houver disponibilidade.

## Canais

| Precisa de… | Use |
| --- | --- |
| Bug, dúvida de uso ou melhoria | [GitHub Issues](https://github.com/KleilsonSantos/kleilson-portfolio/issues) |
| Contribuir com código ou docs | [CONTRIBUTING.md](./CONTRIBUTING.md) · [task-kickoff](./docs/guides/task-kickoff.md) |
| Vulnerabilidade de segurança | [SECURITY.md](./SECURITY.md) — **não** abra issue pública |
| Conduta / comunidade | [CODE_OF_CONDUCT.md](./CODE_OF_CONDUCT.md) |

## Antes de abrir uma issue

1. Confira se já existe issue semelhante.
2. Inclua contexto mínimo: o que tentou, erro/output, SO e versão Node/`pnpm` se for setup.
3. Para setup local, siga [docs/guides/onboarding.md](./docs/guides/onboarding.md) e `.env.example`.

## O que não é suporte deste repo

- Problemas da conta GitHub, Cloudflare, Supabase ou outros serviços de terceiros (use o suporte do fornecedor).
- Pedidos de consultoria ou customização fora do escopo do portfólio.
